Essential R Code Tips for Business Analytics

Master R with these foundational code snippets for data analysis.

1. Loading Data Frames

Loads a dataset from a CSV file into a data frame.

data <- read.csv("file_path.csv")

2. Viewing the First Few Rows

Displays the first six rows of the data frame.

head(data)

3. Splitting Data into Train and Test Sets

Divides data into training (70%) and testing (30%) sets.

set.seed(123)
train_index <- sample(1:nrow(data), 0.7 * nrow(data))
train_data <- data[train_index, ]
test_data <- data[-train_index, ]

4. Fitting a Linear Model

Fits a linear regression model with `y` as the dependent variable and `x` as the independent variable.

model <- lm(y ~ x, data = data)

5. Scatter Plot

Creates a scatter plot of `y` vs. `x`.

plot(data$x, data$y, main = "Scatter Plot", xlab = "X Variable", ylab = "Y Variable")

6. Fitting a Generalized Linear Model (GLM)

Fits a GLM; commonly used for binary outcomes.

model <- glm(y ~ x, family = binomial, data = data)

7. Plotting a Histogram

Generates a histogram of the `x` variable.

hist(data$x, main = "Histogram of X", xlab = "X Variable", col = "lightblue")

8. Creating a Bar Plot

Creates a bar plot of frequencies for a categorical variable.

barplot(table(data$category), main = "Bar Plot of Categories", xlab = "Category", ylab = "Frequency")
