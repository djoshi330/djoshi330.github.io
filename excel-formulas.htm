Excel Formulas for Business Analytics

Essential formulas for business analytics and data analysis in Excel.

1. Descriptive Statistics

AVERAGE

Calculates the mean of a range of numbers.

Formula: =AVERAGE(A2:A10)

MEDIAN

Finds the median (middle value) in a range of numbers.

Formula: =MEDIAN(A2:A10)

STDEV.S

Calculates the standard deviation for a sample of data.

Formula: =STDEV.S(A2:A10)

MIN and MAX

Returns the minimum or maximum value in a range.

Formula: =MIN(A2:A10) or =MAX(A2:A10)

2. Financial Analysis

NPV

Calculates the Net Present Value of an investment based on a discount rate and a series of future cash flows.

Formula: =NPV(discount_rate, cash_flows)

IRR

Returns the Internal Rate of Return for a series of cash flows.

Formula: =IRR(cash_flows)

PMT

Calculates the periodic payment for an annuity based on constant periodic payments and a constant interest rate.

Formula: =PMT(rate, nper, pv)

3. Data Analysis

VLOOKUP

Searches for a value in the first column of a range and returns a value in the same row from another column.

Formula: =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

INDEX and MATCH

Combines INDEX and MATCH for a more flexible lookup.

Formula: =INDEX(return_range, MATCH(lookup_value, lookup_range, 0))

IF

Returns one value if a condition is true, and another value if false.

Formula: =IF(condition, value_if_true, value_if_false)

SUMIF

Sums values in a range based on a condition.

Formula: =SUMIF(range, criteria, [sum_range])
